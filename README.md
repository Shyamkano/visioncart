# 👓 VisionCart Pro

![VisionCart Banner](https://via.placeholder.com/1200x400/0d1117/3b82f6?text=VisionCart+Pro)

VisionCart Pro is a modern, premium e-commerce platform dedicated to eyewear. Designed with a sleek, dark-themed UI and supercharged with a responsive layout, VisionCart provides a seamless browsing and shopping experience across all devices.

## ✨ Features

*   **📱 Fully Responsive Design**: An intuitive app-like bottom navigation for mobile devices and a beautiful glassmorphic top navigation bar for desktop users.
*   **🛒 Seamless Shopping Cart**: Add securely to your cart and checkout effortlessly.
*   **🤖 AI Frame Try-on**: (Coming Soon) Virtual try-on for frames using cutting-edge AI.
*   **🩺 Vision Health & Appointments**: Book appointments with specialized eye care professionals right from the dashboard.
*   **🔐 Secure Authentication**: Fast and safe passwordless login powered by Supabase magic links.
*   **👨‍💼 Admin Dashboard**: Manage inventory, view orders, and organize appointments from a dedicated admin panel.

## 🛠️ Tech Stack

*   **Frontend Ecosystem:** React 18 + Vite
*   **Routing:** React Router DOM
*   **Styling:** Custom CSS (Modern CSS properties, Grid/Flexbox, Glassmorphism)
*   **Icons:** Lucide-React
*   **Backend & Database:** Supabase (PostgreSQL + Auth + Storage)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Node.js (v18 or higher recommended)
*   npm or yarn

### Installation

1.  **Clone the repo**
    ```sh
    git clone https://github.com/your_username/visioncart_pro.git
    cd visioncart_pro
    ```

2.  **Install dependencies**
    ```sh
    npm install
    ```

3.  **Set up Environment Variables**
    Create a `.env` file in the root directory and add your Supabase credentials:
    ```env
    VITE_SUPABASE_URL=your_supabase_project_url
    VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
    ```

4.  **Start the development server**
    ```sh
    npm run dev
    ```

5.  **Open the app**
    Navigate to `http://localhost:5173` in your browser.
